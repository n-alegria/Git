Comandos basicos Git

Comando Uso
git init Inicializa un repositorio
git clone < url > Clona un repositorio remoto
git status Comprueba cambios en el 'workind directory'
git add . Prepara los cambios, los añade al 'staging area'
git commit -m 'mensaje' Añade mensaje capturando los cambios
git push origin master Envía los cambios al repositorio remoto
